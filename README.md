        # kafka — Что такое топик

        Homework template for the Vibe Learn lesson **l1_topic_basics** (Что такое топик).

        ## Task

        Реализуй idempotent Kafka producer на Go. Шлёт N сообщений в топик `orders`. Требования:
- enable.idempotence=true
- ретраит на network error до 3 раз с экспоненциальным backoff
- корректно закрывает соединение на SIGINT
- логирует partition+offset каждого ack
Тесты в template репо проверят: правильность вызова Flush, поведение под симулированной сетевой ошибкой, graceful shutdown.

        ## How to work

        1. The Vibe Learn platform creates a copy of this repo in your GitHub account when you click «Начать домашку» on the lesson.
        2. Clone it locally, complete the TODOs in `main.go`, and push.
        3. CI (`.github/workflows/ci.yml`) runs the tests on every push. The Vibe Learn platform listens to the result via webhook and unlocks the lesson when CI passes.

        ## Local prerequisites

        - Go 1.22+
        - Docker + docker-compose (for the local Kafka cluster, when the lesson requires it)

        ## Running

        ```bash
        go test ./...
        ```
